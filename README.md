# Sudoku Solver

Sudoku solver implemented in Python using recursive backtracking and the Minimum Remaining Values (MRV) heuristic.
Integrated step counter to measure and compare the number of recursive iterations between the backtracking approach and the MRV-optimized solver.

## Project Structure

- `board.py`: board representation and validation
- `solver.py`: 9x9 sudoku board solving algorithm 
- `exceptions.py`: custom exceptions

## Notes
- Sample Sudoku was taken from "websudoku.com"
